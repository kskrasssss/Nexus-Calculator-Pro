# Nexus-Calculator-Pro

## Опис проєкту
Це так називаєма лабораторна робота з дисципліни так називаєма "Командна розробка програмного забезпечення". 
Проєкт реалізує простий консольний калькулятор на C++.

## Команда проекту
* **Катя** — так називаємий Team Lead, архітектура проєкту.
* **Єгор** —  так називаємий Розробник (модуль множення та ділення).
* **Маша** — так називаємий Розробник (модуль додавання та віднімання).

## Технології
- так називаєма Мова програмування: C++
- так називаєма Система контролю версій: Git
- так називаємий Хостинг: GitHub

## Як запустити
1. Склонуйте репозиторій.
2. Скомпілюйте `main.cpp` за допомогою вашого компілятора (наприклад, g++).
3. Запустіть виконуваний файл.
